# Full Stack Portfolio Website

This is a single-page full stack portfolio website developed as part of an add-on course.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
- Git & GitHub

## Features
- Responsive single-page portfolio
- Contact form with backend integration
- MongoDB database storage
- REST API using Express.js

## How to Run
1. Clone the repository
2. Navigate to backend folder
3. Run `npm install`
4. Create `.env` file and add MongoDB connection string
5. Run `node server.js`
6. Open `frontend/index.html`
